# SQL - Structured Query Language 🗄️

Este repositório foi criado com o objetivo de estudar e praticar SQL através do curso oferecido gratuitamente pela [FGV Educação Executiva](https://educacao-executiva.fgv.br/cursos/online/curta-media-duracao-online/sql-structured-query-language). O foco é compreender desde consultas básicas até manipulação e modelagem de dados de forma eficiente.

## 📚 Conteúdo do Curso

Aqui está o mapa de progresso dos tópicos abordados. As caixas de seleção indicam o que já está concluído ou planejado:

### 🔍 Unidade 1 — Consultando Dados

- [ ] **O essencial sobre consultas:** Sintaxe básica do `SELECT`, filtragem com `WHERE` e condições simples.
- [ ] **Operadores especiais e lógicos:** Uso de `BETWEEN`, `IN`, `LIKE`, `IS NULL`, `AND`, `OR` e `NOT`.
- [ ] **Ordenação, limitação e unicidade:** Uso de `ORDER BY`, `LIMIT` e `DISTINCT`.

### 🔗 Unidade 2 — Juntando Dados

- [ ] **Junções internas e naturais:** Combinação de tabelas com `INNER JOIN` e `NATURAL JOIN`.
- [ ] **Junções externas:** Uso de `LEFT JOIN`, `RIGHT JOIN` e `FULL OUTER JOIN`.
- [ ] **Operações de conjuntos:** Combinação de resultados com `UNION`, `INTERSECT` e `EXCEPT`.

### 📊 Unidade 3 — Agrupando os Dados

- [ ] **Funções sumarizadoras:** Uso de `COUNT`, `SUM`, `AVG`, `MIN` e `MAX`.
- [ ] **Agrupando dados:** Agrupamento de registros com `GROUP BY` e filtragem com `HAVING`.
- [ ] **Subqueries:** Consultas aninhadas para análises mais complexas.

### ✏️ Unidade 4 — Manipulando os Dados

- [ ] **Os comandos DML:** Inserção, atualização e remoção de dados com `INSERT`, `UPDATE` e `DELETE`.
- [ ] **Transações:** Controle de transações com `BEGIN`, `COMMIT` e `ROLLBACK`.
- [ ] **Comandos DDL:** Criação, alteração e exclusão de tabelas com `CREATE`, `ALTER` e `DROP`.
